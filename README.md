## 구현 기능

### 기능 1
SFT 데이터를 활용한 QA 모델 fine-tuning & Beam Search, Random sampling 활용한 답변생성
### 기능 2
RM 데이터를 이용한 RewardModel 빌드 & 학습, 생성된 답변에 대한 스코어링
### 기능 3
RM 스코어 결과에 대한 PPO 알고리즘 적용, 모델이 최선의 정책을 수립할 수 있도록 강화학습 적용
